📊 Sales and Profit Dashboard
## Overview

This project is an interactive Sales and Profit Dashboard built using Microsoft Excel. It provides insights into sales, profit, and cost performance across different countries, cities, brands, and sales managers. The dashboard helps users analyze business performance and make data-driven decisions.

## Features
•	Interactive Slicers
•	Country
•	Sales Manager
•	Month
•	Brand

## Dashboard Visualizations

•	KPI Cards for Sales Profit, and Cost
•	Country-wise Sales Performance (Donut Chart)
•	Customer Gender Distribution by Sales Manager (Stacked Column Chart)
•	Cost Distribution by Brand (Area Chart)
•	City-wise Sales & Profit Performance (Horizontal Bar Chart)
•	Sales Current vs Previous Year Trend (Line Chart)
•	Monthly Profit Performance (Waterfall Chart)

## KPI Metrics

•	Total Sales (2024): 1,673,000,

## Project Objective

The main objective of this project is to create an interactive dashboard that helps businesses:

•	Monitor sales, profit, and cost performance
•	Analyze country-wise and city-wise sales
•	Identify top-performing brands and sales managers
•	Compare current year sales with previous year sales
•	Track monthly profit trends
•	Support data-driven business decisions
